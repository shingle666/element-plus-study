## 详细学习内容

### 1. Loading 加载（105分钟）
- 基础加载功能
- 区域加载遮罩
- 全屏加载状态
- 自定义加载图标
- 加载文字提示
- 加载状态管理
- 加载性能优化
- 加载无障碍
- 加载动画定制

### 2. Skeleton 骨架屏（90分钟）
- 基础骨架屏
- 骨架屏模板
- 动画骨架屏
- 自定义骨架屏
- 骨架屏布局
- 骨架屏加载策略
- 骨架屏性能优化

### 3. BackTop 回到顶部（75分钟）
- 基础回到顶部
- 触发条件设置
- 滚动动画配置
- 自定义样式
- 位置控制
- 事件处理
- 移动端适配

### 4. 其他引导组件（45分钟）
- InfiniteScroll 无限滚动
- Anchor 锚点
- 页面加载进度条
- 操作引导提示
