# Basic 基础组件 - Button、Border、Color

## 学习目标

* 掌握 Element Plus 基础组件的使用方法
* 了解 Button 组件的各种类型和属性
* 学习 Border 边框样式的应用
* 掌握 Color 颜色系统的使用
* 理解组件的基本设计原则

#### 1.8 加载状态

```vue
<template>
  <div class="loading-button-demo">
    <el-button type="primary" loading>加载中</el-button>
    <el-button type="primary" :loading="loading" @click="handleClick">
      点击加载
    </el-button>
  </div>
</template>

<script setup>
import { ref } from 'vue'

const loading = ref(false)

const handleClick = () => {
  loading.value = true
  setTimeout(() => {
    loading.value = false
  }, 2000)
}
</script>
```

创建一个包含不同类型、尺寸和状态的按钮组合：

1. 实现一个工具栏，包含主要操作按钮
2. 添加按钮的加载状态和禁用状态
3. 使用按钮组创建分页控件
4. 实现响应式按钮布局

### 练习 2：自定义颜色主题

基于 Element Plus 颜色系统创建自定义主题：

1. 定义自己的品牌色彩
2. 创建自定义的颜色变量
3. 应用到按钮和其他组件
4. 实现暗色主题切换

### 练习 3：边框和阴影效果

练习使用边框和阴影创建视觉层次：

1. 创建不同层级的卡片组件
